🗄️ Step 1 — Create Supabase Account & Project

1Go to supabase.com

Click Start your project → Sign up (free tier is fine)

2Create New Project

Click New Project → Set a project name (e.g. "nippon-pos") → Set a database password → Click Create

⏳ Wait 1-2 minutes for the project to spin up

🗃️ Step 2 — Run the Database Script

3Open SQL Editor

In your Supabase project → Left sidebar → click SQL Editor

4Run the Setup Script

Click New Query → Open the file database-setup.sql → Copy all contents → Paste into editor → Click RUN

✓ You should see "Success. No rows returned"

🔑 Step 3 — Get API Credentials

5Find Your Credentials

In Supabase → Left sidebar → Project Settings → API

You need TWO things:

  • Project URL — looks like https://abcdefgh.supabase.co
  • Anon / Public Key — long string starting with eyJhbGci...
⚠️ Copy the ANON key (not the Service Role key)

🚀 Step 4 — Launch the App

6Open the HTML File

Open nippon-electricals-pos.html in any modern browser (Chrome recommended)

First time it shows the Setup Wizard

7Connect to Database

Step 1 of wizard: Read instructions → Click "Done — Next Step"

Step 2 of wizard: Paste your Project URL and Anon Key → Click Connect & Launch

✓ If successful, it will load the login screen

👤 Step 5 — Login

8Default Login Credentials

❓ Troubleshooting

Login not working?

  • Make sure you ran the SQL script first
  • Try username: admin password: admin123 (exact, case sensitive)
  • Go to Accounts tab → Reconfigure Database if needed

Connection failed?

  • Check you copied the correct Supabase URL (not the Dashboard URL)
  • Make sure you used the anon/public key, not the service role key
  • Confirm the SQL script ran successfully with no errors

Need to reconnect to a different database?

Login → Users tab (admin only) → Scroll down → Click Reconfigure Database Connection
